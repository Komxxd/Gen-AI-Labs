## ğŸ¤– What is a GAN?

A **Generative Adversarial Network (GAN)** is a deep learning framework composed of **two neural networks that compete with each other**:

### ğŸ”¹ Generator (G)

* Takes **random noise** as input
* Tries to **generate images that look real**
* Its goal is to *fool* the discriminator

### ğŸ”¹ Discriminator (D)

* Takes an image as input
* Predicts whether the image is:

  * **Real** (from the dataset)
  * **Fake** (produced by the generator)

---

### ğŸ¯ The Adversarial Game

GAN training is a **zero-sum game**:

* The **Discriminator** wants to correctly identify real vs fake images
* The **Generator** wants to produce images so realistic that the discriminator classifies them as real

As both networks improve, the generator gradually learns to produce **high-quality, realistic images**.
